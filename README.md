# ESP32/8266 Smartmeter Decoder for Kaifa MA309M / Sagemcom T210-D

## **Credits**
Main Code was created by [DomiStyle](https://github.com/DomiStyle) in his project [esphome-dlms-meter](https://github.com/DomiStyle/esphome-dlms-meter).
Thank you!

## Warning
This is a modified version which only supports the Kaifa MA309M and Sagemcom T210-D Smartmeter from Netz NÖ. Also, does not support ESPHome.

## Features
- Reads data from the Smartmeter and sends it to Graphite, a time-series database.
- Displays current values on a tft display
